Privacy Policy for Prayer Times & Adhan Reminder

This Chrome extension respects and protects user privacy. 

1. Data Collection & Usage:
We do not collect, store, log, or transmit any personal data, web browsing history, or tracking identifiers to external servers. 

2. Permissions & Data Handling:
- Geolocation / Location Data: This extension temporarily reads your coordinates solely to process local calculations for prayer timings and Hijri calendars via our declared API endpoints (api.aladhan.com and api.bigdatacloud.net). This processing happens strictly client-side and is never saved.
- Storage Permissions: Utilized natively within the browser profile to remember user interface preferences, such as selected alarms and language configurations.

3. Third-Party Disclosures:
The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements. No user information is shared, transferred, or sold to third parties.
